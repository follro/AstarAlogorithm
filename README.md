# A*経路探索可視化

## 概要
A*アルゴリズムの動作をリアルタイムで視覚的に確認できるツールです。
壁の追加・削除をしながらシミュレーションが可能です。

## 開発背景
アルゴリズムの学習中、実際に動く様子を目で見て理解したいと思い作成しました。

## 使用技術
- C++
- WinAPI

## 機能
- スタート・ゴール地点の設定
- 壁の追加・削除
- A*アルゴリズムによる経路探索のリアルタイム表示
- オープンリスト・クローズドリスト・経路の色分け表示

## 使い方
1. ModeChangeボタンでCreateモードに切り替える
2. Createモードに切り替えると、Start・Destination・Wallボタンが表示される
3. 各ボタンを選択し、クリックで地点を指定する
4. ModeChangeボタンでSimulationモードに切り替える
5. SimulationボタンでA*アルゴリズムの動作を視覚的に確認できる
6. Clearボタンで初期化できる

## 注意事項
- Start・Destination・WallボタンはCreateモード時のみ表示されます
- SimulationはStartとDestinationの両方を設定した状態でないと動作しません
